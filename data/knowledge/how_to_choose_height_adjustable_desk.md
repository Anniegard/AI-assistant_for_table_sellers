# Как выбрать регулируемый стол

Выбор регулируемого стола лучше начинать с трех параметров: рост пользователя, рабочий сценарий и бюджет.

## Шаг 1: Проверьте диапазон высоты

- Для невысоких пользователей важна минимальная высота.
- Для высоких пользователей важна максимальная высота.
- Чем шире диапазон, тем универсальнее модель.

## Шаг 2: Определите размер столешницы

- Один монитор: обычно достаточно 100-120 см ширины.
- Два монитора: комфортнее от 120-140 см.
- Если есть системный блок на столе, лучше брать глубину 70-80 см.

## Шаг 3: Учитывайте нагрузку

Проверьте грузоподъемность, если планируете два монитора, кронштейн, аудиооборудование или тяжелую столешницу.

## Шаг 4: Сценарий использования

Для редкого использования подойдет базовая модель, для ежедневной интенсивной работы лучше выбирать устойчивый двухмоторный стол.
