# Сценарий демонстрации

1. Пользователь открывает Telegram-бота.
2. Нажимает кнопку «Подобрать стол».
3. Отвечает на вопросы по бюджету, росту, количеству мониторов и сценарию использования.
4. Получает 2-3 рекомендации из sample-каталога.
5. Задает вопрос: один или два мотора.
6. Получает ответ из базы знаний.
7. Оставляет заявку с контактами.
8. Менеджер получает заявку через будущий notification/export слой.
9. Бизнес видит, какие данные собрал ассистент и как это ускоряет продажу.
