# Scope MVP

## Входит в MVP

- Telegram-бот.
- Подбор стола по параметрам.
- FAQ-ответы.
- Сбор заявки.
- Локальное сохранение лида.
- Подготовка интерфейса для будущей отправки менеджеру.
- Sample-каталог.
- Sample-база знаний.

## Не входит в MVP

- Полноценная CRM.
- Интеграция в сайт клиента.
- Парсинг всего рынка.
- Сложная админка.
- Production RAG.
- Оплата, корзина, реальные остатки.
- Массовый scraping чужих каталогов.

## Границы итерации

Текущая итерация создает техническую основу проекта и демонстрационные данные. Полноценный Telegram flow и интеграции будут добавляться на следующих этапах.
