# Demo checklist

## Перед демо
- Установлены зависимости: `pip install -r requirements.txt`
- Пройдены проверки: `pytest`, `ruff check .`
- Настроен `.env` с `TELEGRAM_BOT_TOKEN`
- Подготовлен Telegram аккаунт для показа
- Известен путь `LEADS_PATH` для проверки сохраненного JSON

## Smoke-test сценарий
1. Отправить `/start` и проверить кнопки меню: `Подобрать стол`, `Частые вопросы`, `Оставить заявку`, `Демо-режим`.
2. Нажать `Подобрать стол` и пройти все вопросы (бюджет, рост, мониторы, сценарий, моторы).
3. Убедиться, что бот показывает 2-3 рекомендации и кнопку `Оставить заявку по этим вариантам`.
4. Нажать `Оставить заявку по этим вариантам` и заполнить минимум: имя, телефон, город, комментарий.
5. Проверить JSON лида в `LEADS_PATH`: есть `recommended_products` и параметры подбора (`budget`, `height_cm`, `use_case`, `monitors_count`).
6. Если задан `MANAGER_TELEGRAM_CHAT_ID`, проверить сообщение менеджеру с полями: имя, телефон, город, бюджет, рост, сценарий, мониторы, рекомендации, комментарий.
7. Проверить FAQ: спросить по словам `мотор`, `два монитора`, `доставка`, `гарантия`, `материал`.

## После демо
- Показать сохраненный лид в `LEADS_PATH`
- Зафиксировать обратную связь клиента
- Согласовать следующий этап пилота
