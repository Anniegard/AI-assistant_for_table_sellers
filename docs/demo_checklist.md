# Demo checklist

## Перед демо
- Установлены зависимости: `pip install -r requirements.txt`
- Пройдены проверки: `pytest`, `ruff check .`
- Настроен `.env` с `TELEGRAM_BOT_TOKEN`
- Подготовлен Telegram аккаунт для показа

## Во время демо
- `/start` отрабатывает
- Подбор стола проходит до выдачи 2-3 моделей
- FAQ отвечает по базе знаний
- Лид собирается и сохраняется
- При наличии chat id: менеджер получает сообщение

## После демо
- Показать сохраненный лид в `LEADS_PATH`
- Зафиксировать обратную связь клиента
- Согласовать следующий этап пилота
