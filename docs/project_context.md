# Контекст проекта

AI-assistant for table sellers это демонстрационный AI sales assistant для ниши регулируемых по высоте столов, эргономичной мебели и аксессуаров.

## Идея проекта

Собрать MVP, который показывает бизнесу практическую ценность AI-консультанта в продаже технически сложного товара с множеством параметров выбора.

## Целевая аудитория

- Компании, продающие регулируемые столы и эргономичную мебель.
- Руководители отделов продаж и собственники малого и среднего e-commerce бизнеса.
- Команды, которым нужен быстрый proof of value перед внедрением в сайт или CRM.

## Бизнес-гипотеза

Если AI-ассистент берет на себя первичную консультацию, подбор и сбор структурированной заявки, то:

- снижается нагрузка на менеджеров,
- растет скорость обработки входящих запросов,
- повышается качество лидов для отдела продаж.

## Почему сначала Telegram-бот

- Быстрый запуск и короткий time-to-demo.
- Не нужен доступ к сайту клиента на первом этапе.
- Удобный канал для демонстрации сценариев консультации и сбора лидов.
- Проще валидировать гипотезу и собрать обратную связь до более дорогих интеграций.
